To compile and run my  code, I had to first open code blocks and copy the whole code into a new blank page, then saved it as newmain.cpp or it would not have worked. After saving it, I went to the top toolbar and clicked the yellow gear icon which is the build button so  that the computer could check if I made any typing mistakes and change the code into numbers it understands. When it showed me my errors, I corrected them, saved it and  clicked the gear icon again. Finally. When it showed me that there were no existing errors at the bottom, I clicked the green play button next to it to open the black command box where my program started working. The main limitation with my code was that my code didn’t accept more than 30 records.
