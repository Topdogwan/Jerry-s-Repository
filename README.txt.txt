This program is a console-based C++ application designed to manage student academic records using arrays, functions, and pointers.

To compile the program:
1. Open your terminal or command prompt.
2. Navigate to the folder containing the file.
3. Compile using: g++ cs101project.cpp -o program

To run the program:
4. Execute: ./program   (Linux/Mac)
   or program.exe       (Windows)

Features:
- Enter student records (name, ID, scores)
- Compute total scores and assign grades
- Display full class report
- Display class statistics (highest, lowest, average, pass/fail)
- Search for a student by ID

Limitations:
- Maximum number of students is 30
- Data is not saved permanently (program resets when closed)
